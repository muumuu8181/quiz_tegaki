　/ (repo root)

├─ config/                     # 環境/ビルド設定

│  │  ├─ index.html

│  └─ features/                # 垂直スライス（必要になったら増やす）
├─ shared/                     # 共通（utils, i18n, 型）
├─ public/                     # 静的資産
├─ tests/                      # テスト本体のみを置く
│  ├─ unit/  ├─ integration/  └─ e2e/
├─ tools/                      # 旧scripts：開発/運用スクリプト置き場
│  ├─ codegen/   # 例: OpenAPI→型生成
│  ├─ migrate/   # 例: DBマイグレーション起動
